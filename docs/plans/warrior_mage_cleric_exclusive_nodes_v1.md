# 전사·마법사·성직자 전용 노드 트리 v1 spec

작성: 2026-05-16 / 사용자 인터뷰 16/16 반영 / **출시 전 사전 설계 단계**

---

## 1. 배경

- 도적: `north_rogue` zone 28 전용 노드 (47 core + 28 전용 = 75)
- 소환사 v2: `north_summoner_v2_arcane/beast/holy/spirit` 4 zone 89 전용 노드
- 궁수: `archer` zone 95 (전용 클래스, core 별도)
- **미커버**: 전사 / 마법사 / 성직자 — 각 47 core 만 보유

목표: warrior/mage/cleric 도 도적·소환사 패턴으로 전용 노드 트리 추가하여 빌드 다양성 확보.

## 3. 데이터 구조

### 3.1 node_definitions 컬럼 활용
- `class_exclusive`: 'warrior' / 'mage' / 'cleric'
- `zone`: 6개 신규 zone (아래 §4)
- `tier`: 'T1' / 'T2' / 'T3' / 'keystone'
- `cost`: T1=1 / T2=2 / T3=3 / keystone=4
- `prerequisites`: 진입 노드는 core 트리의 끝단 1~2개에 종속
- `effects`: jsonb (기존 컨벤션 그대로)

### 3.2 6 신규 zone
| 직업 | 갈래 1 | 갈래 2 |
|------|--------|--------|
| 전사 | `north_warrior_berserk` (광전사) | `north_warrior_guard` (수호자) |
| 마법사 | `north_mage_flow` (흐르는 마나) | `north_mage_accumulate` (축적 마나) |
| 성직자 | `north_cleric_guard` (수호) | `north_cleric_radiant` (광명) |

### 3.3 갈래 구성 (소형 — 각 12~17 노드)
| Tier | 개수 | 비용 | 누적 비용 |
|------|------|------|-----------|
| T1 (입문) | 6 노드 | 1 | 6 |
| T2 (중급) | 5 노드 + 키스톤 1 | 2 (키스톤 4) | 14 |
| T3 (심화) | 3 노드 + 키스톤 1 | 3 (키스톤 4) | 17 |
| **계** | **15 + 2 키스톤 = 17** | — | **27** |

목표: 한 갈래 완주 = 27포인트 (Lv28+ 부분 투자 / Lv50 풀 투자 가능).

## 4. 클래스별 갈래 상세 (예시 효과)

### 4.1 전사

#### 4.1.1 광전사 (`north_warrior_berserk`)
- **T1×6**: ATK +5/+5/+5/+8/+8/+10 (스탯 패시브)
- **T2×5**: 출혈 데미지 +10/+15/+20/+25/+30%
- **T2 키스톤 — `furor_pulse`**: HP 70% 이하 시 ATK +50% (조건부)
- **T3×3**: CRIT +5/+8/+12, 처치 시 HP 흡수 5%
- **T3 키스톤 — `unbridled_rage`**: 출혈 적 처치 시 다음 3초 ATK +80% (5초 쿨다운)

#### 4.1.2 수호자 (`north_warrior_guard`)
- **T1×6**: HP +50/+50/+50/+80/+80/+100
- **T2×5**: DEF +5/+5/+8/+10/+12, 도발 지속 +1턴
- **T2 키스톤 — `iron_resolve`**: HP 30% 이하 시 받는 데미지 -50%
- **T3×3**: 반격 데미지 +15/+25/+35%, 방벽 흡수량 +20
- **T3 키스톤 — `vengeful_bulwark`**: 피격당하면 다음 평타 +60% (스택 max 3)

### 4.2 마법사

#### 4.2.1 흐르는 마나 (`north_mage_flow`)
- **T1×6**: MAX MANA -5%/-5%/-5%, MANA 회복 +10%/+10%/+15%
- **T2×5**: 스킬 데미지 +5/+8/+10/+12/+15% (마나 부족 시 보너스)
- **T2 키스톤 — `volatile_current`**: MANA 30% 이하 시 스킬 데미지 +50%
- **T3×3**: 쿨다운 -3/-5/-8%
- **T3 키스톤 — `endless_river`**: MANA 0 도달 시 다음 스킬 무료 + 데미지 +70% (10초 쿨)

#### 4.2.2 축적 마나 (`north_mage_accumulate`)
- **T1×6**: MAX MANA +10/+15/+20/+25/+30/+40
- **T2×5**: 마나 풀 비율 비례 스킬 데미지 +N% (mana 80%+ 시 +N%)
- **T2 키스톤 — `crystallize`**: MANA 풀 비례 ATK 보너스 (max +30% at 100%)
- **T3×3**: 마나 소모 -5/-8/-12%, INT +10
- **T3 키스톤 — `mana_burst`**: MANA 100% 도달 시 다음 스킬 +80% (1회/스택)

### 4.3 성직자

#### 4.3.1 수호 (`north_cleric_guard`)
- **T1×6**: HP +60/+60/+60/+80/+80/+100
- **T2×5**: 회복 보너스 +10/+15/+20/+25/+30%, 자가 회복 +N%
- **T2 키스톤 — `divine_aegis`**: 받는 데미지의 30%를 흡수 → MANA 회복으로 변환
- **T3×3**: 방벽 흡수량 +20/+30/+45
- **T3 키스톤 — `unbreakable_faith`**: HP 50% 이하 시 5초간 모든 데미지 -60% (15초 쿨)

#### 4.3.2 광명 (`north_cleric_radiant`)
- **T1×6**: MATK +6/+8/+10/+12/+15/+20
- **T2×5**: 쿨다운 -3/-5/-8%, MANA 효율 +N%
- **T2 키스톤 — `radiant_burst`**: 적 처치 후 다음 3초 모든 스킬 쿨다운 -50%
- **T3×3**: 신성 데미지 +10/+15/+20%
- **T3 키스톤 — `dawn_breaker`**: 광명 키스톤 잠재 + 시그니처 스킬 사용 시 팀 ATK +30% (5초)

## 5. 레이아웃 (position_x / position_y)

도적 north_rogue 패턴 참조 — 캔버스 북측 확장. 6 zone 모두 core 47 끝단에서 위로 뻗는 형태.

기준점 예시 (실제 좌표는 소환사 v2 답습):
- 전사 광전사: 좌측 상단 (-300 ~ +300, -1500 ~ -800)
- 전사 수호자: 좌측 상단 (-300 ~ +300, -1500 ~ -800) — 거울 배치
- 마법사 / 성직자도 각 zone 분리 배치

## 6. 마이그레이션·운영

### 6.1 DB 마이그레이션
- scripts/add_warrior_exclusive_nodes.cjs
- scripts/add_mage_exclusive_nodes.cjs
- scripts/add_cleric_exclusive_nodes.cjs
- 각 스크립트는 17 노드 × 2 갈래 = 34 노드 INSERT (총 102 신규 노드)

### 6.2 기존 캐릭터
- 자동 무료 리스펙 제공 (1회) — `paragon_v2` 진입 시 패턴 동일
- 점수: 기존 투자된 core 47 그대로 유지 (별도 영향 없음)

### 6.3 PvP 밸런스 모니터링
- 출시 1주 후 데이터 추적: warrior/mage/cleric 의 PvP 승률 변화
- 키스톤 조건 발동률 추적 (`unbridled_rage` 등) → 너무 잦으면 cooldown 증가 검토
