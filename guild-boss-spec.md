# 길드 보스 — 설계 스펙

> 작성일: 2026-04-17
> 모델: 요일제 7종 보스, 입장키 일일 2개, 입장당 데미지 컷 보상

---

## 1. 핵심 컨셉

- **길드 전용 콘텐츠** — 길드 가입 필수 (미가입자 입장 불가)
- **개별 도전, 개별 보상** — 길드원 각자 자기 키로 입장, 자기 데미지 측정, 자기 상자 수령
- **HP 무제한 보스** — 처치 개념 없음, 입힌 데미지가 곧 점수
- **요일제 7종 로테** — 매일 다른 보스, 빌드 다양성 자극
- **길드 단위 시너지** — 일일 누적 데미지로 글로벌 상자 잠금 해제 (전 길드원 보상)

---

## 2. 입장 시스템

| 항목 | 값 |
|---|---|
| 입장키 지급 | 길드원 각자 일일 2개 |
| 초기화 시각 | 매일 KST 00:00 (보스 변경 / 키 초기화 / 누적 초기화 모두 동일 시각) |
| 1키 = | 1회 입장 (별개 데미지 측정) |
| 입장 시간 제한 | 무제한 (유저 수동 퇴장만 가능) |
| 키 단위 | 캐릭터당 (계정당 X — 부캐 운영자도 정상 사용) |
| 데미지 캡 | 캡 없음 |
| 미사용 키 이월 | 없음 (자정 초기화 시 소실) |
| 전투 방식 | 기존 자동 사냥과 동일 — 스킬 자동 사용, 게이지 충전, 접두사 / 노드 효과 그대로 적용 |
| 플레이어 사망 시 | **부활 없음 — 입장 즉시 종료**. 사망 직전까지 누적된 데미지는 보존되며, 해당 데미지 컷에 맞는 상자 지급 |
| 오프라인 진행 | **비활성** — 로그아웃 시 자동 퇴장 (오프라인 방치 폭딜 방지) |
| 데미지 집계 DB 저장 | 10초마다 flush (실시간 UI는 클라이언트 캐시) |

## 3. 보상 구조

### 3.1 입장당 1상자 (가장 높은 컷만)

| 입힌 데미지 | 수령 상자 |
|---|---|
| 10억 이상 | 황금빛 상자 |
| 5억 이상 ~ 10억 미만 | 은빛 상자 |
| 1억 이상 ~ 5억 미만 | 구리 상자 |
| 1억 미만 | 없음 |

### 3.2 황금빛 상자

**확정 보상 (매번 지급)**:
- 강화 성공 스크롤 1개
- EXP 두루마리 (현 레벨 EXP의 5퍼센트)
- 골드 5,000,000
- 길드 보스 메달 50개
- 부스터 5종 패키지 (EXP/골드/드랍/공격력/HP 각 1시간, 효과 +50퍼센트)

**잭팟 보상 (각 1퍼센트 독립 굴림)**:
- 아이템 품질 재굴림권
- 접두사 수치 재굴림권
- 창고 슬롯 영구 +1
- 유니크 무작위 추첨권

### 3.3 은빛 상자

**확정 보상**:
- 골드 2,500,000
- 길드 보스 메달 15개
- EXP 두루마리 (현 레벨 EXP의 3퍼센트)
- 부스터 5종 패키지 (각 1시간, 효과 +25퍼센트 — 황금의 절반)

**잭팟 보상**:
- 아이템 품질 재굴림권 (확률 1퍼센트)
- 접두사 수치 재굴림권 (확률 5퍼센트)

### 3.4 구리 상자

**확정 보상**:
- 골드 1,000,000
- 길드 보스 메달 5개
- EXP 두루마리 (현 레벨 EXP의 1퍼센트)
- 부스터 택1 (EXP/골드/드랍 중 1종, 1시간, 효과 +10퍼센트)

**잭팟 보상**:
- 접두사 수치 재굴림권 (확률 1퍼센트)

## 4. 보스 메커닉 (공통)

### 4.1 약점 시간대
- 매 30분마다 30초간 약점 노출
- 약점 시간 동안 모든 데미지 ×3
- 길드 채팅에 자동 알림 ("XX 보스 약점 노출! 30초간!")

### 4.2 누적 디버프 (집단 시너지)
- 길드원이 보스를 때릴 때마다 보스에 누적 디버프
- 1만 타격 = 보스 방어 -1퍼센트
- 캡 -50퍼센트 (50만 타격 시 도달)
- 늦게 입장하는 길드원이 더 효율 ↑

### 4.3 임계값 첫 통과 보너스
- **본인 기준** — 한 입장 세션 안에서 본인이 각 임계값을 처음 통과하는 순간 메달 보너스 즉시 지급
- 10억까지 도달한 입장은 3개 모두 획득 (5 + 15 + 30 = 누적 +50)
  - 1억 첫 통과: +5 메달
  - 5억 첫 통과: +15 메달
  - 10억 첫 통과: +30 메달

### 4.4 메커닉 우선순위
- **약점 시간대**(×3)가 다른 메커닉보다 우선
- 약점 시간대 동안:
  - 시계태엽 거인의 HP 회복 **일시 중지**
  - 차원 지배자의 ATK/MATK 면역 페이즈 **일시 해제** (순수 폭딜 창구 확보)
- 누적 디버프(4.2)는 약점 시간대와 독립적으로 계속 누적 / 적용

## 5. 요일제 보스 7종

각 보스는 공통 메커닉 + 고유 특성을 가짐. 모두 HP 무제한.

> **능력치 기준선**: DEF / MDEF / DODGE는 현재 라이브 서버 **상위 10퍼센트 캐릭 스탯을 기준**으로 산출. "명중 빌드 아닌 일반 캐릭이 3~5퍼센트만 빗맞는" 선이 표준. 고유 특성을 적용한 보스에서만 수치 과장.

### 5.1 월요일 — 강철의 거인 (Iron Colossus)
- **외형**: 풀플레이트 거인, 양손 도끼
- **고유 특성**: DEF 매우 높음 (3000+) / MDEF 보통 (1000)
- **약점/유리 빌드**: 방관 (def_pierce_pct + def_reduce_pct), 마법사
- **불리 빌드**: 순수 물리 ATK 빌드는 방관 없으면 데미지 큰 폭 감소
- **운영 의도**: 캐릭 빌드 다양성 강제

### 5.2 화요일 — 광속의 환영 (Lightspeed Wraith)
- **외형**: 반투명 그림자, 빠른 잔상
- **고유 특성**: 회피율 매우 높음 (50퍼센트), 명중 디버프 무시
- **약점/유리 빌드**: 명중 (accuracy 스탯, 정밀한 계열 접두사)
- **불리 빌드**: 명중 낮은 빌드는 절반이 빗나감
- **운영 의도**: 명중 스탯 가치 부각

### 5.3 수요일 — 화염의 군주 (Flame Sovereign)
- **외형**: 용암 기반 마왕
- **고유 특성**: 화염 속성 데미지 100퍼센트 면역 / 빙결 속성 받는 데미지 +50퍼센트
- **약점/유리 빌드**: 빙결 속성 스킬 (마법사 냉기 창 / 빙결 감옥 / 절대 영도 등)
- **불리 빌드**: 화염 위주 마법사
- **운영 의도**: 원소 속성 시스템 활용

### 5.4 목요일 — 그림자 황제 (Shadow Emperor)
- **외형**: 어둠 망토 왕좌
- **고유 특성**: 모든 도트(독/화상/출혈) 100퍼센트 면역
- **약점/유리 빌드**: 직접 데미지 위주 (전사, 마법사 직타 스킬)
- **불리 빌드**: 도적, 마법사 도트 빌드는 큰 폭 약화
- **운영 의도**: 도트 의존도 측정 / 직타 빌드 보상

### 5.5 금요일 — 시계태엽 거인 (Clockwork Titan)
- **외형**: 거대한 시계 메커니즘 골렘
- **고유 특성**: 매 60초마다 누적 데미지의 30퍼센트 회복 (페이즈 회복)
- **약점/유리 빌드**: 폭딜 빌드 (60초 안에 회복분보다 더 입혀야 임계값 통과)
- **불리 빌드**: 느린 누적 도트 빌드
- **운영 의도**: 폭딜 vs 지속딜 구분, 빠른 클리어 인센티브

### 5.6 토요일 — 천공의 용 (Skybound Dragon)
- **외형**: 거대한 다색 용
- **고유 특성**: 매 입장 시 약점 원소 랜덤 (화염/빙결/번개/대지/신성/암흑 중 1)
- **약점/유리 빌드**: 입장 후 표시되는 약점 원소에 맞춰 빠르게 대응
- **운영 의도**: 다양성, 운빨 요소

### 5.7 일요일 — 차원의 지배자 (Dimensional Overlord)
- **외형**: 변형하는 보이드
- **고유 특성**: 30초 주기로 ATK 면역 ↔ MATK 면역 페이즈 교대 (공중 표시 카운트다운)
- **약점/유리 빌드**: 양손 빌드 (전사/도적+마법사 혼성 어렵지만 ATK/MATK 양립 가능한 캐릭이 유리)
- **운영 의도**: 단일 데미지 타입 빌드 견제, 보스의 가장 어려운 페널티

## 6. 길드 단위 시너지 (글로벌 보상)

길드 내 모든 멤버의 일일 데미지 합산이 임계값 도달 시, 전 길드원에게 자동 지급.

| 누적 데미지 | 보상 (전 길드원에게) |
|---|---|
| 100억 | 미니 글로벌 상자 — 골드 1,000,000 + 메달 5개 |
| 500억 | 미디엄 글로벌 상자 — 강화 스톤 30개 + 메달 15개 |
| 1,000억 | 메가 글로벌 상자 — 접두사 수치 재굴림권 1개 + 메달 30개 |
| 5,000억 | 길드 전체 24시간 EXP/골드/드랍 +25퍼센트 버프 |

자정에 누적 초기화. 임계값 도달 시 **즉시** 전 길드원 우편함으로 자동 지급 (자정 정산 X).

## 7. 운영 / 장기 시스템

### 7.1 길드 보스 메달 상점

메달은 캐릭터별 누적 보관. 상점은 카테고리별로 분류되며 구매 제한(일 / 주 / 월 / 계정)으로 과다 구매 방지.

#### 대형 — 5,000 ~ 20,000 메달 (월 1~2개 구매 목표)

| 상품 | 가격 | 구매 제한 | 비고 |
|---|---|---|---|
| 유니크 장비 무작위 추첨권 | 8,000 | 주 1회 | 캐릭 레벨 ±10 풀에서 무작위 1개 |
| T3 접두사 보장 추첨권 | 15,000 | 월 1회 | 1 옵션 T3 확정 굴림 |
| 영구 스탯 묘약 세트 | 12,000 | 월 1회 | HP / ATK / MATK 영구 +3 (캐릭 캡 50) |
| 창고 슬롯 영구 +3 | 10,000 | 계정 5회 한도 | 영구 확장 |
| 호칭 "길드영웅" 영구 부여권 | 7,000 | 계정 1회 | 영구 호칭 |
| 3옵 보장 굴림권 | 15,000 | 월 1회 | 장비 1개에 강제 3접두사 옵션 부여 |

#### 중형 — 500 ~ 3,000 메달 (주간 핵심 성장)

| 상품 | 가격 | 구매 제한 | 비고 |
|---|---|---|---|
| 접두사 수치 재굴림권 | 1,200 | 주 3회 | 수치 재굴림 |
| 강화 성공 스크롤 | 800 | 주 5회 | 1회 사용 |
| 유니크 조각 | 2,000 | 주 3회 | 3개 모아 유니크 합성 |
| 부스터 6시간 패키지 (5종 +50%) | 3,000 | 주 5회 | EXP / 골드 / 드랍 / 공격력 / HP 동시 |

#### 소형 — 50 ~ 300 메달 (일상 소모품)

| 상품 | 가격 | 구매 제한 | 비고 |
|---|---|---|---|
| 골드 묶음 (100만) | 100 | 일 3회 | 즉시 골드 |
| 고급 포션 10개 (HP 80%) | 50 | 일 5회 | 소모품 |
| 부스터 1시간 택1 | 150 | 일 3회 | EXP / 골드 / 드랍 중 1종 |
| EXP 두루마리 (현 레벨 1%) | 200 | 일 2회 | 즉시 EXP |
| PvP 공격권 +1 | 100 | 일 2회 | PvE↔PvP 교차 |
| 일일임무 즉시 완료권 | 250 | 일 1회 | 임무 1개 클리어 |

#### 길드 단위 — 길드장만 구매

| 상품 | 가격 | 구매 제한 | 비고 |
|---|---|---|---|
| 길드 전체 24시간 +25% 버프 | 5,000 | 주 1회 | EXP / 골드 / 드랍 |
| 디버프 캡 -50 → -70% 24시간 | 3,000 | 일 1회 | 다음 24시간 길드 보스 전용 |
| 길드 창고 슬롯 +1 | 8,000 | 월 2회 | 길드 창고 시스템 신설 요건 |
| 길드 명성 +1,000 즉시 | 2,000 | 주 2회 | 길드 레벨업 가속 |

### 7.2 길드 보스 도감
- 잡은 보스 7종 기록
- 각 보스의 베스트 데미지(개인 / 길드 단위) 기록
- 최초 임계값 통과자(10억) 명예의 전당 표시

### 7.3 주간 결산
- 매주 일요일 23:00 KST 자동 집계
- 길드별 일일 누적의 7일 합계로 1위 ~ 3위 결정
- 1위 길드: 서버 공지 + 7일간 "왕좌" 호칭 / 모든 길드원에게 메달 200개
- 2위 길드: 메달 100개
- 3위 길드: 메달 50개

## 8. UI / UX

### 8.1 길드 화면
- 신규 탭 "길드 보스" 추가

### 8.2 보스 메인 페이지
- 오늘의 보스 (요일제) 카드 표시 — 외형 / 이름 / 고유 특성 짧은 설명
- 남은 입장키 (예: 2/2, 1/2, 0/2)
- 길드 일일 누적 데미지 미터 + 4개 임계값 게이지 (100억 / 500억 / 1000억 / 5000억)
- 길드원 일일 데미지 랭킹 보드 (TOP 10)
- "입장하기" 버튼 (키 보유 시 활성)

### 8.3 입장 / 전투 화면
- 보스 HP: "∞" 표시
- 내 누적 데미지 카운터 (실시간)
- 다음 상자까지 남은 데미지 미터 (1억 / 5억 / 10억)
- 약점 시간대 카운트다운 (다음 약점까지 N초)
- 약점 시간 동안 화면 깜빡임 + "약점 노출!" 효과
- 누적 디버프 게이지 (보스 방어 감소율 표시)

### 8.4 종료 / 결과 화면
- 입힌 총 데미지 표시
- 받은 상자 (황금/은/구리/없음)
- **사망으로 종료된 경우** — "사망" 표시 + 입장 직전까지 누적된 데미지로 상자 판정
- 상자 개봉 애니메이션 + 보상 목록
- **부스터 계열 보상**은 인벤토리에 **아이템 형태로 저장** → 유저가 원할 때 우클릭 사용 (쌓기 가능)
- **유니크 무작위 추첨권** 당첨 시 — 캐릭터 레벨 ±10 범위 유니크 풀에서 무작위 1개 지급
- 길드 채팅 자동 알림 ("플레이어 이름이 황금빛 상자 획득!")

## 9. 데이터 모델 (DB 스키마 초안)

```sql
-- 길드 보스 정의
CREATE TABLE guild_bosses (
  id SERIAL PRIMARY KEY,
  name VARCHAR(40) NOT NULL,
  weekday INT NOT NULL,  -- 0=월 ~ 6=일
  description TEXT,
  base_def INT NOT NULL DEFAULT 0,
  base_mdef INT NOT NULL DEFAULT 0,
  base_dodge INT NOT NULL DEFAULT 0,
  element_immune VARCHAR(20),
  element_weak VARCHAR(20),
  weak_amp_pct INT DEFAULT 0,
  dot_immune BOOLEAN DEFAULT FALSE,
  hp_recover_pct INT DEFAULT 0,
  hp_recover_interval_sec INT DEFAULT 0,
  random_weakness BOOLEAN DEFAULT FALSE,
  alternating_immune BOOLEAN DEFAULT FALSE
);

-- 캐릭터 일일 키 / 누적 데미지
CREATE TABLE guild_boss_daily (
  character_id INT REFERENCES characters(id) ON DELETE CASCADE,
  date DATE NOT NULL,
  keys_remaining INT NOT NULL DEFAULT 2,
  daily_damage_total BIGINT NOT NULL DEFAULT 0,
  PRIMARY KEY (character_id, date)
);

-- 입장 기록 (입장 1회 = 1행)
CREATE TABLE guild_boss_runs (
  id BIGSERIAL PRIMARY KEY,
  character_id INT REFERENCES characters(id),
  guild_id INT REFERENCES guilds(id),
  boss_id INT REFERENCES guild_bosses(id),
  started_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  ended_at TIMESTAMPTZ,
  total_damage BIGINT NOT NULL DEFAULT 0,
  reward_tier VARCHAR(10), -- 'gold' | 'silver' | 'copper' | NULL
  first_threshold_passed VARCHAR(10) -- 'gold' | 'silver' | 'copper' | NULL
);

-- 길드 일일 누적
CREATE TABLE guild_boss_guild_daily (
  guild_id INT REFERENCES guilds(id),
  date DATE NOT NULL,
  total_damage BIGINT NOT NULL DEFAULT 0,
  global_chest_milestones INT NOT NULL DEFAULT 0, -- bitmask: 1=100억, 2=500억, 4=1000억, 8=5000억
  PRIMARY KEY (guild_id, date)
);

-- 메달 보유
ALTER TABLE characters ADD COLUMN guild_boss_medals INT NOT NULL DEFAULT 0;
```

## 11. 튜닝 파라미터 (메달 경제)

- 황금 상자 확정 메달 50 + 10억 첫통과 보너스 50 = **최대 100 / 입장**
- 일일 2키 기준 최대 **200 메달 / 캐릭 / 일**
- 주간 최대 약 1,400 / 캐릭 / 주
- 월간 최대 약 6,000 / 캐릭 / 월
- 글로벌 상자 누적 보상 (+30, +15, +5 등)은 추가 가산
- **메달 상점 가격은 위 획득량을 기반으로 역산 — 대형 상품(예: 유니크 조각)은 5,000~20,000, 중형은 500~2,000, 소형 소모품은 50~300 정도로 설계 예정**
